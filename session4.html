Sets in Python

  • A set is an unordered collection of unique and immutable elements.
  • Sets are mutable so we can add or remove elements.
  • Cannot contain mutable items like lists or dictionaries.

Set Implementation 

# Creating a set with duplicate elements
numbers = {1, 2, 3, 4, 4, 5}
print("Set (duplicates removed):", numbers)  # Output: {1, 2, 3, 4, 5}

## Creating a set from a list using set()
fruits = set(["apple", "banana", "apple", "orange"])
print("Set from list:", fruits)  # Output: {'apple', 'banana', 'orange'}

# Empty set (must use set(), not {})
empty_set = set()
print("Empty Set:", empty_set)

# Add elements to a set
fruits.add("grapes")
print("After adding grapes:", fruits)

# Remove element using remove() - raises error if not present
fruits.remove("banana")
print("After removing banana:", fruits)

# ✅ Set Operations
set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}

# Union - combines elements
print("Union:", set_a | set_b) 

# Intersection - common elements
print("Intersection:", set_a & set_b)

# Set of squares
squares = {x**2 for x in range(0,11)};
print("Squares Set:", squares)

Immutable and Mutable in Python

Feature Mutable Immutable
Definition Objects whose values can be changed after creation Objects whose values cannot be changed once created
Examples list, dict, set, bytearray int, float, str, tuple, bool, frozenset
Can modify contents? ✅ Yes ❌ No (new object is created on change)
Memory Behavior Modified in place Creates a new object

Example

#string and int are immutable data types

# Assigning a string value to the variable 'username'
username = "shree"
print(username)  # Output: shree

# Reassigning a new value to the same variable 'username'
username = "Ram"
print(username)  # Output: Ram

# Assigning an integer value to variable 'x'
x = 10

# Copying the value of x into variable 'y'
y = x
print(x, y)  # Output: 10 10
# At this point, both x and y hold the value 10, but they are independent integers.

# Changing the value of x
x = 25
print(y)  # Output: 10
# y still holds the old value 10, because integers are immutable and assignment creates a new reference.

# Now assigning y to reference the new value of x
y = x
print(y)  # Output: 25
# y now holds the updated value of x, which is 25.

Conditional Statements in Python

Conditional statements help in decision-making by executing certain blocks of code based on specified conditions.

  • Simple if: Executes a block of code only if the condition is true.
  • if-else: Provides an alternative block if the condition is false.
  • if-elif-else: Checks multiple conditions in sequence, and executes the block for the first true condition.
  • Nested if: One if or if-else inside another if block; used for multi-level decision-making.
  • match: The match statement is Python's version of a switch-like pattern matching system. It allows you to match a variable's value against a series of patterns (cases) and execute corresponding blocks of code.

if..else Implementation

# 1. Simple if
age = 20
if age >= 18:
    print("You are eligible to vote.")  # This line runs because age is 20

# 2. if-else
marks = 55
if marks >= 40:
    print("You passed the exam.")  # This line runs because marks >= 40
else:
    print("You failed the exam.")

# 3. if-elif-else
score = 85
if score >= 90:
    print("Grade: A")
elif score >= 75:
    print("Grade: B")  # This line runs because score is between 75 and 89
elif score >= 60:
    print("Grade: C")
else:
    print("Grade: D")

# 4. Nested if
number = 10
if number > 0:
    print("Positive number")
    if number % 2 == 0:
        print("Even number")  # This line runs because 10 is even
    else:
        print("Odd number")
else:
    print("Non-positive number")

#5. match statement
command = "start"

match command:
    case "start":
        print("System is starting...")
    case "stop":
        print("System is stopping...")
    case "pause":
        print("System is paused.")
    case _:
        print("Unknown command.")

#ternary expression
age = 16
print("Adult" if age >= 18 else "Minor")
