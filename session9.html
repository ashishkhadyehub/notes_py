User Input

In console-based applications, user input allows the program to interact with the user by accepting input data during execution.

  • Python uses the input() function to get input from the user via the keyboard.
  • input() always returns a string.

Example

name = input("Enter your name: ")
print("Hello,", name)

# Taking two numbers as input
num1 = float(input("Enter the first number: "))
num2 = float(input("Enter the second number: "))

# Adding the numbers
sum_result = num1 + num2

# Displaying the result
print("The sum is:", sum_result)

Error Handling

Error handling in Python is used to gracefully manage errors and exceptions that may occur during the execution of a program — instead of crashing the program, we handle the error and decide what to do next.

Example

try:
    a = int("abc")  # will raise ValueError
except (ValueError, TypeError) as e:
    print("Error occurred:", e)
finally:
    print("Execution Completed")

pass keyword

The pass keyword in Python is used as a placeholder when a statement is required syntactically, but you don't want to write any code there yet.

Example

x = 10
if x > 5:
    pass  # Logic will be added later
else:
    print("x is 5 or less")

Decorators

A decorator is a function that takes another function as an argument and adds some functionality to it without modifying its structure.

It is often used to wrap another function to extend its behavior.

Why use decorators?

  • Code reusability
  • Cleaner and more readable code
  • Separation of concerns (e.g., logging, access control, validation)

Example

# Decorator function
def log_decorator(func):
    def wrapper():
        print("Function is being called...")
        func()
        print("Function has finished executing.")
    return wrapper

# Using the decorator
@log_decorator
def say_hello():
    print("Hello, World!")

# Call the decorated function
say_hello()

#decorator with argument
def greet_decorator(func):
    def wrapper(name):
        print("Preparing to greet...")
        func(name)
        print("Greeting complete.")
    return wrapper

@greet_decorator
def greet(name):
    print(f"Hello, {name}!")

greet("Shree")

pip

pip is the official package manager for Python and is used to install and manage third-party Python libraries from PyPI (Python Package Index).

Why Use pip?

  • To install new libraries and tools (e.g., requests, Django, numpy)
  • To upgrade or remove packages
  • To manage dependencies in a project using requirements.txt
Command Description Example
pip install package_name Install a package pip install requests
pip install package==version Install a specific version pip install Django==4.2.0
pip install -U package Upgrade a package pip install -U pandas
pip uninstall package Uninstall a package pip uninstall flask
pip list List all installed packages pip list
pip show package Show details of an installed package pip show numpy
pip freeze Output installed packages in format for requirements.txt pip freeze > requirements.txt
pip install -r requirements.txt Install packages from a requirements.txt file pip install -r requirements.txt

Example

Install Package: rich
pip install rich

The rich library allows you to build beautiful terminal applications with rich formatting

from rich.console import Console
from rich.table import Table

# Create a console object
console = Console()

# Create a table
table = Table(title="Top Python Libraries")

# Add columns
table.add_column("Library", justify="left", style="cyan", no_wrap=True)
table.add_column("Use Case", style="magenta")
table.add_column("Popularity", justify="right", style="green")

# Add rows
table.add_row("NumPy", "Numerical Computing", "★★★★★")
table.add_row("Pandas", "Data Analysis", "★★★★★")
table.add_row("Rich", "Terminal Output Formatting", "★★★★☆")
table.add_row("Requests", "HTTP for Humans", "★★★★☆")

# Print the table
console.print(table)

SQLite3

  • SQLite is a lightweight, file-based relational database.
  • It's built-in with Python (no need to install separately).
  • Great for small apps, desktop tools, and prototyping.
Concept Code Example Explanation
import import sqlite3 Imports the built-in sqlite3 module to use SQLite database functions in Python.
connect() conn = sqlite3.connect('mydata.db') Creates a connection to the SQLite database file. If the file doesn't exist, it will be created.
cursor() cursor = conn.cursor() Creates a cursor object used to execute SQL commands.
execute() cursor.execute("SQL statement") Executes an SQL command like CREATE, INSERT, SELECT, UPDATE, DELETE, etc.
commit() conn.commit() Saves (commits) changes made to the database. Required after INSERT, UPDATE, DELETE.
fetchall() rows = cursor.fetchall() Retrieves all rows from the result of a SELECT query.
fetchone() row = cursor.fetchone() Retrieves the next row of a query result. Returns None when no more data is available.
close() conn.close() Closes the database connection. Always do this to free up resources.
