Lists in Python

A list is a built-in data type in Python used to store multiple items in a single variable.

  • Ordered: Items have a defined order.
  • Mutable: Items can be changed after the list is created.
  • Heterogeneous: Can store elements of different data types (e.g., integers, strings, even other lists).
  • Indexed: Access items using index positions starting from 0.

List Implementation

# Creating a List
fruits = ["apple", "banana", "cherry","mango"]
numbers = [10, 20, 30, 40]
mixed = ["hello", 42, True]

#Accessing List Items

print(fruits[0])     # apple
print(fruits[-1])    # cherry (last item)
print(fruits[2:3])

#modifying item
fruits[1] = "orange"
print(fruits)        # ['apple', 'mango', 'cherry']

fruits.append("dragon fruit")         # Adds at end
fruits.insert(1, "kiwi")        # Adds at index 1

fruits.remove("apple")          # Removes by value
fruits.pop()                    # Removes last item
del fruits[0]                   # Removes by index

for fruit in fruits:
    print(fruit,end=" |")

fruits.sort()       # Sorts alphabetically
fruits.reverse()    # Reverses order
print(len(fruits))  # Number of items

copyfruits = fruits.copy()  # Copy the list

print(copyfruits)

if "mango" in fruits:
    print("mango is in the skill list.")

Dictionary in Python

A dictionary in Python is a collection of key-value pairs. It is mutable, and indexed by keys (not positions).

  • Key-Value Pairs Data stored in pairs like "key": value
  • Mutable : Can change, add, or delete items
  • Unique Keys : Keys must be unique, values can repeat
  • Indexed by Keys, accessed using keys, not by numeric index like lists
  • Maintains insertion order starting from Python 3.7+

Dictionary Implementation

# ‚úÖ Creating a dictionary with key-value pairs
skills = {
    "front_end": "HTML",
    "back_end": "C#",
    "database": "SQL",
}

# üîç Accessing dictionary values
print(skills["front_end"])     # Output: HTML
print(skills.get("back_end"))  # Output: C#

# ‚úèÔ∏è Updating a value
skills["front_end"] = "JavaScript"

# üîÅ Looping through dictionary (keys only)
for skill in skills:
    print(skill, skills[skill])

# ‚úÖ Checking if a key exists
if "front_end" in skills:
    print("Front-end skill is available.")

# üìè Length of dictionary
print(len(skills))  # Output: Number of key-value pairs

# ‚ûï Adding a new key-value pair
skills["Sharing"] = "GitHub"
print(skills)

# ‚ùå Removing a specific key
skills.pop("Sharing")

# ‚ùå Removing the last inserted item
skills.popitem()

# ‚ùå Deleting a key-value pair by key
del skills["front_end"]

# ‚ùå Clearing all items from dictionary
skills.clear()
print(skills)  # Output: {}

# üîÅ Copying a dictionary
skills_copy = skills.copy()

# üì¶ Nested Dictionary Example (real-life use case)
products = {
    "Garment": {
        'Mens Wear': ['Shirts', 'Trousers'],
        'Womens Wear': ['Dresses', 'Skirts']
    },
    "Electronics": {
        'Mobile Phones': ['Smartphones', 'Feature Phones'],
        'Laptops': ['Gaming Laptops', 'Ultrabooks']
    }
}

# üîç Accessing nested dictionary values
print(products)
print(products["Garment"])                    # Output: Sub-dictionary under 'Garment'
print(products["Garment"]["Mens Wear"])       # Output: ['Shirts', 'Trousers']

# üîÅ Dictionary Comprehension: Square of numbers 1‚Äì10
squares = {x: x**2 for x in range(1, 11)}
print(squares)

Tuples in Python

A tuple is an ordered, immutable collection of items.

  • Ordered: Elements maintain the sequence in which they were defined.
  • Immutable: Once created, items in a tuple cannot be modified, added, or removed.
  • Heterogeneous: Tuples can store elements of different data types (e.g., int, str, float).
  • Iterable: Tuples can be iterated using loops like for and comprehensions.
  • Efficient: Tuples are faster and use less memory compared to lists.

Tuple Implementation

# Creating a tuple
my_tuple = (10, 20, 30, 40)
print("Tuple:", my_tuple)

# Tuples can contain different data types
mixed_tuple = (1, "Hello", 3.14, True)
print("Mixed Tuple:", mixed_tuple)

# Accessing elements using indexing
print("First element:", my_tuple[0])
print("Last element:", my_tuple[-1])

# Slicing a tuple
print("Sliced Tuple [1:3]:", my_tuple[1:3])

# Length of the tuple
print("Length of tuple:", len(my_tuple))

# Iterating through a tuple
print("Elements in my_tuple:")
for item in my_tuple:
    print(item)

name_age = ("Shree", 30)
name, age = name_age
print("Name:", name)
print("Age:", age)

# Nested tuple
nested = ((1, 2), (3, 4))
print("Nested Tuple:", nested)
print("Access nested item:", nested[1][0])  # Output: 3

# Tuple with one element (must include a comma)
single_element = (5,)
print("Single-element Tuple:", single_element)
print("Type of single_element:", type(single_element))  # Should be tuple

# min and max 
t = (34, 78, 12, 99, 56)
print("Min:", min(t))
print("Max:", max(t))


# Repetition of tuple
repeat_tuple = ("Python",) * 3
print("Repeated Tuple:", repeat_tuple)
