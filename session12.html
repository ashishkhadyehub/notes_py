API Handling in Python

API (Application Programming Interface) allows different software systems to communicate.

requests Package

  • requests is a popular, easy-to-use HTTP library in Python.
  • It allows you to send HTTP requests and handle responses.
  • We can install requests package by using below command : 
    pip install requests

Example

import requests

# 1. Take PIN code from user
pin_code = input("Enter a 6-digit Indian PIN code: ")

# 2. API URL
url = f"https://api.postalpincode.in/pincode/{pin_code}"

# 3. Send GET request
response = requests.get(url)
data = response.json()

# 4. Parse and display results
if data[0]['Status'] == 'Success':
    post_offices = data[0]['PostOffice']
    print(f"\nDetails for PIN code {pin_code}:")
    for office in post_offices:
        print("-" * 40)
        print(f"Post Office  : {office['Name']}")
        print(f"District     : {office['District']}")
        print(f"State        : {office['State']}")
        print(f"Branch Type  : {office['BranchType']}")
else:
    print("Invalid PIN code or no data found.")
