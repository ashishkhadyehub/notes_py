Introduction

Python is a high-level, interpreted, general-purpose programming language known for its simplicity and readability. It is designed to be easy to learn and is widely used in web development, data science, artificial intelligence etc.

Key Features of Python

  • Simple & Readable syntax
  • Interpreted language (no compilation needed)
  • Cross-platform support (Windows, Linux, macOS)
  • Extensive Standard Library/li>
  • Supports multiple programming paradigms (Procedural, Object-Oriented, Functional)
  • Rapid Application Development
  • Excellent Libraries and Frameworks

Brief History of Python

Year Milestone
1989 Guido van Rossum began developing Python as a hobby project.
1991 First official release: Python 0.9.0
2000 Python 2.0 released with features like garbage collection and list comprehensions.
2008 Python 3.0 released — major upgrade with better Unicode support and cleaner syntax.
2020 Python 2 officially discontinued; Python 3 becomes the standard.
Now Python 3 widely used in data science, machine learning, automation, web development, and more.

Python Implementations

Implementation Description
CPython The standard and most commonly used Python implementation, written in C. Default version from python.org.
PyPy High-performance Python implementation with JIT (Just-In-Time) compilation. Much faster for many tasks.
Jython Python implemented in Java. Allows use of Java libraries and runs on the Java Virtual Machine (JVM).
IronPython Python for the .NET framework, written in C#. Integrates with C# and .NET libraries.
MicroPython Lightweight Python for microcontrollers and IoT devices like Raspberry Pi Pico, ESP32, etc.

Working of Python Application

Working of Python Console Application
Step Description
1. Writing the Code The programmer writes Python code and saves it in a .py file.
2. Compilation Python automatically compiles the source code into bytecode (.pyc) and stores it in the __pycache__ folder.
3. Interpretation The Python Virtual Machine (PVM) executes the bytecode line-by-line.
4. Output Display The result of the execution is shown in the console/terminal.
5. Error Handling Python immediately displays any errors (syntax, name, etc.) when they occur.
Key Concepts
Line-by-line Execution Python executes the code sequentially, which simplifies debugging.
Interpreter-Based No separate compilation step is required like in Java or C++.
Bytecode & __pycache__ Bytecode files improve performance by caching compiled code.
Cross-platform Python runs the same code on multiple operating systems.
Error Visibility Errors are shown immediately, helping in fast development.

Python in Shell

  • The Python shell is an interactive command-line interface that allows you to write and execute Python code one line at a time.
  • You can enter the shell by typing: 
    python

Example of Using Python Shell

>>> print("Hello from Python Shell")
Hello from Python Shell

>>> a = 5
>>> b = 10
>>> a + b
15

>>> import hello
Welcome to Python Programming
Getting Started
>>> hello.printdata("Python in Shell")
Python in Shell
>>>

Notes

  • Use the Python shell for testing logic and understanding behavior.
  • Use script files (.py) for actual program development and reuse.
  • Use exit() or press Ctrl + Z (Windows) / Ctrl + D (Linux/Mac) to exit the shell.
