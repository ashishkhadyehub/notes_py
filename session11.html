Python Virtual Environment

A virtual environment is an isolated Python environment where:

  • You can install packages independently of the global Python.
  • Projects can have separate dependencies and package versions.
  • It helps prevent version conflicts between different Python projects.

Why Use Virtual Environments?

Reason Explanation
Dependency Isolation Each project can use different versions of libraries without conflict.
Avoid Polluting System Global Python remains clean as packages are installed only in the virtual environment.
Easy Project Management Environments can be created, removed, and shared easily using requirements.txt.
Better Deployment Ensures consistent setup across development, staging, and production systems.
Testing Multiple Versions Allows you to test your code against different versions of libraries or Python itself.

Example

  1. Create Virtual Environment 
    python -m venv env
  2. Activate the Environment 
    env\Scripts\activate
  3. To exit the virtual environment 
    deactivate
  4. Install Packages Inside Virtual Environment 
    pip install package-name
  5. Export installed packages 
    pip freeze > requirements.txt
  6. Install from requirements.txt 
    pip install -r requirements.txt

Python Open Source Code

Python is an open-source language, and its interpreter, standard library, and many third-party packages are available for anyone to explore and contribute.

Why Explore Open Source Python Code?

  • Understand how professional Python code is written.
  • Learn real-world use of modules, classes, and functions.
  • Improve your problem-solving and debugging skills.
  • Contribute to projects and grow your developer profile (e.g., GitHub).
  • Learn how large applications are structured and maintained.

Where to Explore Python Source Code

Source Description

Anaconda

Anaconda is a free and open-source Python distribution used for scientific computing, data science, machine learning, and big data processing. It simplifies package management and deployment.

Download Anaconda

Key Features of Anaconda

Feature Description
Pre-installed Libraries Comes with 250+ scientific libraries like NumPy, Pandas, Matplotlib, etc.
Package Manager Uses conda, which is more powerful and flexible than pip.
Virtual Environments Allows isolated environments for different projects using conda create.
Cross-platform Works on Windows, macOS, and Linux.
GUI Support Includes Anaconda Navigator, a user-friendly interface for managing environments and packages.
Jupyter Notebook Integrated with Jupyter for interactive and data-driven development.
Data Science Friendly Optimized for data science, machine learning, and big data applications.

Common Anaconda Commands

Command Description Example
conda list Lists all installed packages in the current environment. conda list
conda create --name py310 python=3.10 Creates a new environment with the specified python version. conda create -n myenv numpy
conda activate env_name Activates the specified environment. conda activate myenv
conda deactivate Deactivates the current active environment. conda deactivate
conda info --envs Lists all environments available on your system. conda info --envs
conda install package_name Installs a package into the current environment. conda install pandas
conda update conda Updates the Conda package manager itself. conda update conda
conda update package_name Updates a specific package. conda update numpy
conda remove package_name Removes a specific package from the environment. conda remove matplotlib
conda env remove -n env_name Deletes an entire Conda environment. conda env remove -n myenv

Anaconda Navigator

Anaconda Navigator is a graphical user interface (GUI) included with the Anaconda distribution. It allows users to manage packages, environments, and launch applications without using command-line tools.

Miniconda

Miniconda is a minimal, lightweight version of the Anaconda distribution. It includes only the Conda package manager and Python, without the pre-installed data science libraries that come with full Anaconda.

Jupyter Notebook

Jupyter Notebook is an open-source web-based interactive environment where you can create and share documents that contain:

  • Live code
  • Equations
  • Visualizations
  • Text (Markdown)

It's especially popular in data science, machine learning, and education.

Key Features of Jupyter Notebook

Feature Description
Interactive Code Execution Write and run code in small cells, allowing real-time results and iterative development.
Multi-language Support Supports Python, R, Julia, and other languages using different kernels.
Rich Text & Markdown Use Markdown to format text, insert headers, lists, links, and LaTeX equations.
Data Visualization Inline visualization support for libraries like matplotlib, seaborn, and plotly.
Notebook File Format (.ipynb) Saves work in shareable JSON-based .ipynb files.
Integration with Libraries Easily integrates with pandas, NumPy, scikit-learn, TensorFlow, etc.
Web Interface Runs in a web browser for a clean, interactive user experience.

Example

Create Jupyter Notebook file (.ipynb)
Markdown Cell
### Calculate the Area of a Circle

This notebook calculates the area of a circle using the formula:

$$
\text{Area} = \pi \times r^2
$$

We will ask the user for the radius and compute the area.
Code Cell
import math

# Input from user
radius = float(input("Enter the radius of the circle: "))

# Calculate area
area = math.pi * radius ** 2

# Display result
print(f"The area of the circle with radius {radius} is {area:.2f}")
