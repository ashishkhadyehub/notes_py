SQLite3 Example

import sqlite3

db_name = "test.db"

conn = sqlite3.connect(db_name)

cursor = conn.cursor()

cursor.execute('''create table if not exists users(
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT,
        contact TEXT,
        city TEXT
               )
''')

cursor.execute("insert into users (name,contact,city) values  (?,?,?)",('shree','8850024342','Kolhapur'))
cursor.execute("insert into users(name,contact,city) values (?,?,?)",('ram','8850024343','Pune'))

conn.commit()

cursor.execute("select * from users")
rows=cursor.fetchall()

for user in rows:
    print(f"Name:{user[1]}, Contact:{user[2]}, City:{user[3]} ")

conn.close()

__name__ Attribute

  • In Python, every module has a special built-in attribute called __name__
  • When a Python file is run directly, __name__ is set to "__main__".

Example

try:
    def greet():
    print("Hello, Python!")

if __name__=="__main__":
    greet()

Tkinter

  • Tkinter is the built-in library in Python for creating Graphical User Interfaces (GUIs).
  • It allows you to build windows, buttons, labels, text boxes, and more.
  • No installation is needed. It's included with Python.

Common Tkinter Widgets

Widget Purpose Example
Label Display text tk.Label(root, text="Hello")
Button Clickable button tk.Button(root, text="Click")
Entry Single-line text input tk.Entry(root)
Text Multi-line text input tk.Text(root)
Checkbutton Checkbox tk.Checkbutton(root, text="Yes")
Radiobutton Select one from multiple tk.Radiobutton(...)
Listbox List of items tk.Listbox(root)
Frame Group other widgets tk.Frame(root)

Example

import tkinter as tk

root = tk.Tk()             # Create the main window
root.title("My App")       # Set the window title
root.geometry("300x200")   # Set the window size

root.mainloop()            # Keep the window open
