Loops in Python

Loops are used to execute a block of code repeatedly as long as a condition is met or over a sequence

Loop Type / Keyword Description
for loop Used for iterating over a sequence (like list, tuple, string).
while loop Executes as long as the condition is true.
break Exits the loop prematurely.
continue Skips the current iteration and continues with the next one.
range() Generates a sequence of numbers to use with for loop.

Loops Implementation 

# -------------------------------
# 1. for loop example
# -------------------------------

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)  # Prints each fruit in the list

print("---------------------------")

# -------------------------------
# 2. while loop example
# -------------------------------

count = 1
while count <= 5:
    print("Count:", count)  # Prints count from 1 to 5
    count += 1  # Increment count by 1

print("---------------------------")

# -------------------------------
# 3. Using break in a for loop
# -------------------------------

for i in range(10):
    if i == 5:
        break  # Exit loop when i is 5
    print("Break example:", i)

print("---------------------------")

# -------------------------------
# 4. Using continue in a for loop
# -------------------------------

for i in range(5):
    if i == 2:
        continue  # Skip iteration when i is 2
    print("Continue example:", i)

print("---------------------------")

# -------------------------------
# 5. Using range(start, stop, step)
# -------------------------------

for i in range(1, 10, 2):  # Start at 1, stop before 10, step by 2
    print("Range with step:", i)

Functions in Python

A function is a block of organized, reusable code used to perform a single, related action. Functions help break a program into smaller, modular parts.

Function Implementation

# 1. Basic Function (No Parameters, No Return)
def greet():
    print("Hello, welcome to Python!")

greet()  # Output: Hello, welcome to Python!

# ---------------------------------------------

# 2. Function with Parameters
def greet_user(name):
    print("Hello", name)

greet_user("Shree")  # Output: Hello Shree

# ---------------------------------------------

# 3. Function with Return Value
def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print("Addition:", result)  # Output: Addition: 8

# ---------------------------------------------

# 4. Function with Default Parameter
def welcome(name="User"):
    print("Welcome", name)

welcome()          # Output: Welcome User
welcome("Mr ABC")    # Output: Welcome ABC

# ---------------------------------------------

# 5. Function with *args (Variable-length non-keyword arguments)
def print_total(*numbers):
    total = sum(numbers)
    print("Total:", total)

print_total(1, 2, 3, 4)  # Output: Total: 10

# ---------------------------------------------

# 6. Function with **kwargs (Variable-length keyword arguments)
def print_profile(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_profile(name="Shree", age=30, location="Kolhapur")
# Output:
# name: Shree
# age: 30
# location: Kolhapur

# ---------------------------------------------

# 7. Lambda Function to compute the cube of a number
cube = lambda x: x ** 3
print("Cube of 3:", cube(3))  # Output: Cube of 3: 27

Scopes and Closure

Scope refers to the region of a program where a variable is recognized and accessible.

Scope Type Description Example
Local Scope Variables declared inside a function and only accessible within that function. 
def my_func():
    x = 10  # Local variable
    print("Local:", x)
my_func()
Enclosing Scope Applies to nested functions. Inner functions can access variables from the outer function. 
def outer():
    msg = "Hello"
    def inner():
        print("Enclosing:", msg)
    inner()
outer()
Global Scope Variables declared at the top level of a script or module, accessible everywhere. 
x = 5  # Global variable
def display():
    print("Global:", x)
display()
Built-in Scope Names that are preassigned in Python like print(), len(), etc. 
print("Built-in function")  # No need to define print()

Scopes Implementation

# GLOBAL SCOPE
x = "Global Scope"

def show_global():
    print("Global Scope Value:", x)

show_global()


# LOCAL SCOPE
def local_example():
    y = "Local Scope"  # This variable is only accessible inside this function
    print("Local Scope Value:", y)

local_example()
# print(y)  # This would raise an error


# ENCLOSING SCOPE
def outer():
    msg = "Enclosing Scope"  # Enclosing variable

    def inner():
        print("Enclosing Scope Value:", msg)  # Accesses variable from outer()

    inner()

outer()


# BUILT-IN SCOPE
length = len([1, 2, 3])  # len() is a built-in function
print("Built-in len():", length)

Closure

A closure is a function object that remembers values from its enclosing lexical scope, even after the outer function has finished executing.

Closure Implementation

# Define the outer function that takes one argument
def outer_func(msg):
    # Define an inner function that uses the 'msg' variable from the outer function
    def inner_func():
        # This line will print the value of 'msg'
        print(f"Message: {msg}")
    
    # Return the inner function, not calling it yet
    return inner_func

# Call outer_func with a message and assign the returned inner_func to a variable
my_printer = outer_func("Hello Closure")

# Now call the returned function
my_printer()  # Output: Message: Hello Closure

# Even though 'outer_func' has finished executing,
# 'inner_func' still remembers the value of 'msg'.
